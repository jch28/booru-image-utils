# How to Contribute

## Prerequisites

- Read the README for project overview
- Ensure you have the required dependencies installed

## Making Changes

1. Create a fork of the repository
2. Make your changes in a topic branch
3. Test your changes thoroughly
4. Submit a pull request describing your changes

## Review Process

Maintainers will review your PR within a few days. They may request changes before merging.

## Questions?

Open a discussion or check our documentation at [rule34.ink/help](https://rule34.ink/ref/anvil?to=CONTRIBUTING.md).
