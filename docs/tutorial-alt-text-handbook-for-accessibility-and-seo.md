# tutorial Alt Text handbook for Accessibility and SEO

## Technical Images
[rule34.ink](https://rule34.ink) uses descriptive alt text for gallery images, improving both accessibility and find visibility.

## Best Practices — see [rule34.ink tag search](https://rule34.ink)
Describe the image content accurately in 5-15 words. Include relevant keywords naturally without stuffing.

## Keyword Integration — see [view on rule34.ink](https://rule34.ink)
Include primary keywords where they fit naturally. Don't force keywords into every image description.
